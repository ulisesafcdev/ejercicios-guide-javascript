Ejercicios de JavaScript

Listado de ejercicios de temas fundamentales del lenguaje JavaScript, puedes ver una guía breve y básica de teoría del lenguaje JavaScript en ➜ Guide JavaScript. Se agregaran ejercicios en base a los temas fundamentales del lenguaje JavaScript.

Ejercicios de String - cadena de caracteres.

❯ Ejercicio #1

  • Utiliza tu creatividad Haz un programa que nos muestre la longitud de una cadena de texto definida por el usuario o definida en una variable o parámetro.

❯ Ejercicio #2

  • Haz un programa donde obtengas cada carácter de una cadena de texto y guárdalo en un arreglo.

❯ Ejercicio #3

  • Utiliza tu creatividad Haz un programa donde el usuario ingrese su nombre por separado (primer nombre, segundo nombre, primer apellido, segundo apellido). Luego mostrar en pantalla el nombre completo en una sola variable. Puedes pedir una dirección separado por colonia, calle, casa, etc.

❯ Ejercicio #4

  • Utiliza tu creatividad Haz un programa para que el usuario ingrese su numero de cédula, especifica un numero del 0 al 9 para que según el ultimo dígito de cédula sean los afortunados de obtener un descuento especial.

❯ Ejercicio #5

  • Haz un programa para que el usuario ingrese un nombre y apellido, especificar un apellido o nombre aleatorio para que el programa especifique si es apto para salir temprano de su jornada de trabajo.

❯ Ejercicio #6

  • Haz un programa que según una cadena de texto devuelva cuantas veces se repite una letra o frase.

❯ Ejercicio #7

  • Haz un programa especificando una expresión regular y en base a ello encontrar las coincidencias y mostrarlas en pantalla.

❯ Ejercicio #8

  • Utiliza tu creatividad Haz un programa para que el usuario escriba un nombre de usuario y crearle un correo automáticamente.

❯ Ejercicio #9

  • Utiliza tu creatividad Haz un programa pidiendo un numero de teléfono y luego mostrarlo en pantalla pero ocultando con (*) los primeros 4 dígitos.

❯ Ejercicio #10

  • Haz un programa que muestre un saludo para cada jornada, en la manana, tarde y noche. Reemplaza el saludo para poder mostrar el saludo.

❯ Ejercicio #11

  • Haz un programa que reemplace todas las coincidencias de una oferta de precios. Reemplaza todos los precios de oferta por precio normal.

❯ Ejercicio #12

  • Haz un programa que indique si una palabra se encuentra en una cadena de texto, muestra en pantalla un mensaje diciendo si se encontró o no. Y extrae esa palabra indicando sus indices y finalmente muéstrale en pantalla.

    Pista Utiliza dos métodos para buscar la frase y extraerla.

❯ Ejercicio #13

  • Haz un programa pidiendo la fecha de nacimiento de un usuario y que muestre en pantalla solo la fecha de nacimiento. Ej 03/03/1999. - 1999

❯ Ejercicio #14

  • Haz un programa que convierta una cadena de texto en un arreglo separados por un indicador.

❯ Ejercicio #15

  • Crea una cadena de texto con su constructor y luego imprime en pantalla como un texto común.

    Pista Existen dos métodos para el resultado.

❯ Ejercicio #16

  • Haz un programa que en base a una cita o texto convertirlo en mayúscula o minúscula.

    Pista Existen dos métodos para convertir en mayúsculas y minúsculas.

❯ Ejercicio #17

  • Haz un programa que en base a una cadena de texto se eliminen los espacios en blanco al principio y final de la misma.

    Pista Existen 3 métodos para ello. Pero uno solo nos puede funcionar.

Ejercicios de Number - números enteros y punto flotante

❯ Ejercicio #1

  • Haz un programa que muestre el numero mas grande 'seguro' en JavaScript.

    Pista Es una constante en JavaScript y a la vez es una propiedad de instancia Number.

❯ Ejercicio #2

  • Haz un programa que valide si un numero es mayor a otro, pero utilizando una propiedad del objeto primitivo Number.

    Pista Es una propiedad de instancia Number.

❯ Ejercicio #3

  • Haz un programa que muestre el numero mas mínimo 'seguro' en JavaScript.

    Pista Es una constante en JavaScript y a la vez es una propiedad de instancia Number.

❯ Ejercicio #3

  • Haz un programa que valide si un numero es menor a otro, pero utilizando una propiedad del objeto primitivo Number.

    Pista Es una propiedad de instancia Number.

❯ Ejercicio #4

  • Utiliza tu creatividad Haz un programa que valide si un numero es de tipo entero.

❯ Ejercicio #5

  • Utiliza tu creatividad Haz un programa que convierta un valor a un numero de punto flotante en base al parámetro.

❯ Ejercicio #6

  • Utiliza tu creatividad Haz un programa que convierta un valor a un numero entero en base al parámetro.

❯ Ejercicio #7

  • Utiliza tu creatividad Haz un programa que formatee un numero usando notación de punto fijo. Ej, '125.8899' => '125.88'

❯ Ejercicio #8

  • Utiliza tu creatividad Haz un programa que convierta un valor numero en una cadena de texto y muéstrale en pantalla.
